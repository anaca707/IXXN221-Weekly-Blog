Coding Lecture 

This lecture covered part 1 of CSS grids, which talks about divs and flexboxes. Grids consist of a grid container and grid items that can be arranged in rows and columns. Divs help us manage where things go on our website and group elements together. There are 2 main types of divs, an ID and a class. IDs you don’t repeat in your code whereas classes do.  Flexboxes on the other hand help us lay out and distribute items within containers. This is helpful because it allows us to ensure that content can be viewed nicely even when the screen size is different.   


These concepts and coding are still a bit confusing to me, but they will definitely be useful once I start coding for my assignment and use them to lay out my elements.
